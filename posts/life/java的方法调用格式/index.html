Java的方法调用格式

2023-04-06 · 1 min · 木木

Java中的方法调用格式是对象名.方法名(参数列表)。其中，对象名可以是类的对象、类的静态成员或者类本身。如果是调用静态方法，则可以使用类名.方法名的形式。参数列表是方法需要的参数，可以是多个参数，用逗号分隔开。如果方法不需要参数，则可以省略参数列表。在调用方法时，需要注意方法的访问权限和作用域。如果方法是私有的，则只能在类内部进行调用，而不能在其他类中进行调用。

示例：

public void setName(String name)

{``

if(name.contains("小")) return;

this.name = name;

}

其中name是对象名，方法是contains()，参数列表则是"小"

这段代码是一个Java方法，用于设置对象的名称。如果名称中包含字符“小”，则该方法会直接返回，不会设置对象的名称。否则，该方法会将输入的名称赋值给对象的名称属性。这段代码主要是为了避免对象名称中包含“小”字，以避免一些不必要的问题。
